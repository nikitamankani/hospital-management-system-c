# Hospital Management System (C)

A console-based hospital management system written in C that allows hospital staff to manage patient records.

The program provides a simple menu-driven interface for adding, viewing, and managing patient data.

---

## Features

- Add new patient records
- View existing patient records
- Store patient information using file handling
- Simple menu-based interface
